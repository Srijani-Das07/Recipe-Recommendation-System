##  How to Run

### Requirements:
- A C++ compiler (like GCC)
- [Code::Blocks IDE](http://www.codeblocks.org/) or any C++ IDE

### Steps:
1. Clone or download the repository
2. Open main.cpp in Code::Blocks or your IDE
3. Compile and run the program
